## Featured Project

### [Turbine Predictive Maintenance](https://github.com/BortPablo/turbine_predictive_maintenance)
**End-to-end ML system for wind turbine component failure prediction** — extending MSc thesis with production-grade engineering.

- **Problem:** Predict temperature anomalies in critical components (generator, gearbox, hydraulic group, transformer, bearings) to enable condition-based maintenance
- **Approach:** Sliding-window time-series forecasting with XGBoost, residual-based anomaly detection, cost-benefit optimization. Bayesian hyperparameter tuning via scikit-optimize. MLflow for experiment tracking & model registry.
- **Results:** Quantified economic benefit per turbine/component (e.g., **€79K** gearbox bearing savings, **€59K** hydraulic oil savings annually). Containerized with Docker, orchestrated on Kubernetes (GKE).
- **Stack:** Python, XGBoost, scikit-optimize, MLflow, Docker, Kubernetes (GKE), Pandas/Polars, Jupyter
